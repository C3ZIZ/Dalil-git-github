**1. دليل Git البسيط (Git - The simple guide)**

* **الوصف:** عبارة عن صفحة ويب واحدة فقط تشرح دورة العمل الأساسية.
* **لماذا هو مفيد؟** خالي تماماً من الحشو؛ يعطيك الأمر البرمجي ووظيفته مباشرة في سطر واحد. الجميل أنه يدعم اللغة العربية، ويعتبر نموذجاً ممتازاً للاستلهام منه في طريقة سرد الأوامر داخل ملف `README`.
* **كيف تبحث عنه:** اكتب في بحث جوجل: `Git the simple guide Arabic`

**2. توثيق Git في موسوعة حسوب (Hsoub Wiki)**

* **الوصف:** مرجع عربي برمجي مقسم بشكل ممتاز جداً.
* **لماذا هو مفيد؟** يشرح المفاهيم (مثل الفرق بين المسار النسبي والمطلق، والفروع) والأوامر بلغة عربية تقنية دقيقة ومختصرة. كل أمر له صفحة قصيرة تشرح استخدامه بأمثلة نصية واضحة.
* **كيف تبحث عنه:** اكتب في بحث جوجل: `موسوعة حسوب Git`

**3. دروس Atlassian Git (Atlassian Git Tutorials)**

* **الوصف:** مقالات نصية تعليمية من الشركة العريقة Atlassian (متاح بالإنجليزية).
* **لماذا هو مفيد؟** يُعتبر من أفضل المصادر المكتوبة على الإنترنت لشرح المفاهيم المعقدة قليلاً (مثل الفروع Branches وطلبات الدمج PRs). نصوصه مختصرة جداً ومدعومة برسومات توضيحية بسيطة تثبت المعلومة، ويمكنك اقتباس طريقة شرحهم للمفاهيم.
* **كيف تبحث عنه:** اكتب في بحث جوجل: `Atlassian Git tutorial`

**4. التوثيق الرسمي (GitHub Docs - Quickstarts)**

* **الوصف:** التوثيق الرسمي من منصة GitHub نفسها.
* **لماذا هو مفيد؟** قسم "Hello World" وقسم "Quickstart" مكتوبان كنصوص بخطوات مرقمة (1، 2، 3..) وبشكل مختصر جداً. ممتاز جداً لفهم كيف تشرح منصة GitHub نفسها للمستخدمين المبتدئين.
* **كيف تبحث عنه:** اكتب في بحث جوجل: `GitHub Docs Hello World`

**5. موقع (Learn Git Branching)**

* **الوصف:** موقع يعتمد على قراءة نصوص تعليمية قصيرة جداً (متاح بالعربية) ثم تطبيق الأمر.
* **لماذا هو مفيد؟** الشرح النصي فيه قبل كل خطوة يعتبر من أذكى وأبسط الشروحات المكتوبة لمفهوم الفروع (Branches) وكيفية التنقل بينها.
